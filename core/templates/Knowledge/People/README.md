# People

Use this folder for durable notes about PMs, engineering counterparts, stakeholders, and important client contacts.

## What belongs here
- direct-report PM notes
- recurring stakeholder context
- coaching themes and support needs
- open loops that should survive beyond one meeting
- follow-up context for 1:1s or escalations

## Suggested structure for each note
- Role / team / context
- Current responsibilities
- Recent wins
- Open loops
- Coaching themes or support needs
- Next follow-up or next 1:1 topics
- Related feature requests or meetings

## Guidance
- Base notes on concrete evidence, not vague impressions.
- Reinforce wins as clearly as blockers.
- If the same theme repeats across multiple weeks, keep it visible here and bring it into weekly review.
