## Maintenance Rule

When the strategy changes:
1. update the relevant internal strategy notes
2. refresh decision-support notes if the change affects protection, dependency, or intervention logic
3. update the shareable outputs
4. refresh the status-output index if a new dated output is added

## Source of Truth Rule

The standalone HTML command center is a source/reference artifact.
The Personal OS markdown system is the durable source of truth.
